# 图片重命名配置说明

## 配置位置

`Settings` → `Tools` → `Markdown Image Kit` → `图片处理` → `图片重命名`

## 占位符说明

| 占位符                  | 说明    | 示例                                  |
|----------------------|-------|-------------------------------------|
| `${datetime:format}` | 日期时间  | `${datetime:yyyyMMdd}` → `20250101` |
| `${string:length}`   | 随机字符串 | `${string:6}` → `AbCdEf`            |
| `${number:length}`   | 随机数字  | `${number:6}` → `123456`            |
| `${filename}`        | 原文件名  | `${filename}` → `screenshot`        |

**注意**：文件扩展名会自动保留，无需在模板中指定。

## 常用模板示例

### 1. 保持原文件名

```
${filename}
```

效果：`screenshot.png` → `screenshot.png`

### 2. 日期 + 随机字符串

```
${datetime:yyyyMMdd}_${string:6}
```

效果：`image.png` → `20250101_AbCdEf.png`

### 3. 日期时间 + 随机字符串

```
${datetime:yyyyMMddHHmmss}_${string:4}
```

效果：`photo.jpg` → `20250101123456_AbCd.jpg`

### 4. 日期 + 原文件名

```
${datetime:yyyy-MM-dd}_${filename}
```

效果：`diagram.png` → `2025-01-01_diagram.png`

### 5. 前缀 + 日期 + 随机数字

```
img_${datetime:yyyyMMdd}_${number:6}
```

效果：`photo.png` → `img_20250101_123456.png`

### 6. 纯随机字符串（6位）

```
${string:6}
```

效果：`image.png` → `AbCdEf.png`

### 7. MIK前缀 + 随机字符串（兼容旧版）

```
MIK-${string:6}
```

效果：`image.png` → `MIK-AbCdEf.png`

## 日期格式说明

常用的日期时间格式：

| 格式                  | 说明      | 示例                  |
|---------------------|---------|---------------------|
| `yyyyMMdd`          | 年月日（紧凑） | `20250101`          |
| `yyyy-MM-dd`        | 年月日（横线） | `2025-01-01`        |
| `yyyyMMddHHmmss`    | 年月日时分秒  | `20250101123456`    |
| `yyyy-MM-dd_HHmmss` | 混合格式    | `2025-01-01_123456` |

## 推荐配置

**日常使用**：`${datetime:yyyyMMdd}_${string:6}`

- 便于按日期排序
- 避免文件名冲突
- 长度适中

**技术文档**：`${datetime:yyyy-MM-dd}_${filename}`

- 保留原文件名便于识别
- 添加日期前缀便于管理

**快速截图**：`${string:6}`

- 简短随机名称
- 快速生成
